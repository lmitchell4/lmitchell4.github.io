Tournament Organizer

This is a Python program for organizing multi-player tournaments using the Swiss pairing system and recording the results in a PostgreSQL database. I wrote a custom algorithm that generates matches between players of comparable skill levels while preventing rematches between players. The program can also accommodate an odd number of players, in which case a different player is assigned a bye in each round of the tournament.

Alpha Blog

This is a template for a blogging website that allows users to create, edit, and delete blog posts, as well as comment on and rate posts. It uses the Google App Engine Python Standard Environment and is designed to be hosted on the Google Cloud Platform. The website has a responsive design based on the Bootstrap framework.

Tasker

Tasker is a simple Android app for creating to-do lists. This was my first time playing with Android Studio so much time was spent learning to use activities, bundles, and intents.

Alphabet Game

This is a game designed to help young children learn the alphabet. Players are shown a sequence of consecutive letters with some letters missing, and need to decide which letters are missing. I created this game as a Swing application because I enjoy programming in Java and figuring out how to best structure the classes in my programs.

Sturgeon

This is a statistical model designed to estimate the brood year strength of white sturgeon in the San Francisco Estuary using age and length data. It is written in R, and has a Shiny interface that allows the user to try out different inputs. For example, you can choose which years to include or exclude in the model, change the binning scheme of the fish length data, and choose the type of growth model to apply.

Recharge

This was a collaborative USGS research project resulting in the development of a method for estimating water-table recharge from discrete rain events (link to article). I wrote an R program implementing the method that is available upon request from the lead author.

Matching Game

This is a simple web-based matching game where the user clicks on tiles with hidden images and tries to match up tiles with the same image. I created this game because I really wanted to play with JavaScript and jQuery and see if I could carry out all the features I had envisioned – from the animated message board in the top right corner, to the speed controls that allow you to choose how long mismatched tiles remain visible, to the hint button that helps you out when you can’t find a match.
